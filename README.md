A robust, accessible sign-up form built with HTML5, CSS, and Vanilla JavaScript.

### Key Features
* **Real-time Validation:** Immediate user feedback with debouncing.
* **Auto-Save:** Persists draft data to `localStorage` to prevent data loss.
* **Smart Logic:** Dynamic "Company" field & phone number normalization (+358).
* **Security:** Hidden honeypot field for bot detection.
* **Simulation:** Mock API submission using `fetch` and `async/await`.

### Tech Stack
* **Frontend:** Semantic HTML5, CSS Variables, Vanilla JS.
* **Accessibility:** WCAG-compliant with ARIA attributes.

### Learning Outcomes
Focus on DOM manipulation, asynchronous data handling, browser storage, and Git version control.
